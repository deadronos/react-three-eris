# pinball-micro

A tiny physics-driven pinball table example for `react-three-eris`.

## Controls

- Left flipper: `Z` or `←`
- Right flipper: `/` (Slash) or `→`
- Plunger: hold `Space` to charge, release to launch
- Reset: `R`

## Run

From the repo root:

- `pnpm --filter pinball-micro dev`
